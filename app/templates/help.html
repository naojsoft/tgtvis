Text

Format
- Name: no space allowed btw characters, short name preferred
- RA: hhmmss.s* or hh:mm:ss.s*
- DEC: ddmmss.s* or dd:mm:ss.s*
- Format error will be indicated if any
- Max num of targets is 35-ish

Ope

Upload ope/prm files
- You can choose multiple ope files
- If an ope includes the line *load xxxx.prm, upload a prm file as well
- Max num of targets is 35-ish

Laseer

Upload a laser collision file
- The file must be a file provided by a suport scientist.

Interactive Methods

Legend
- Turn on/off entries by clicking.
Buttons for Laser collision time
- Turn on/off laser collision time.
Active Tools
- Pan Tool : Pan the plot by left-dragging a mouse.
- BoxZoom Tool : Zoom the plot bounds.
- WheelZoom Tool : Zoom the plot in and out
- Undo Tool : Restore previous state of the plot.
- Save Tool : Save a PNG image of the plot

Chrome, Firefox tested
